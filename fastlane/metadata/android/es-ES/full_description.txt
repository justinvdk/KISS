Inicie aplicaciones desde la pantalla de inicio escribiendo algunas letras.

La aplicación se adapta a tus hábitos, para que encuentres lo que buscas aún más rápido.
Sus datos nunca dejarán su dispositivo y puede esperar un ahorro de batería.
Su dispositivo será más rápido, en comparación con el uso de un iniciador complejo lleno de funciones inútiles.
KISS ocupa solo 250 KB y nunca se conecta a Internet.

Esta aplicación utiliza la API de accesibilidad de Android para habilitar la función "Doble toque para bloquear la pantalla". No se accede a ningún dato ni se almacena en el proceso.
